# Analisis-Ventas-Superstore
## Objetivo del Proyecto
Transformar datos crudos en insights estratégicos para optimizar la rentabilidad de una empresa de retail, identificando fugas de capital y productos estrella.

## Stack Tecnológico
SQL Server: Limpieza de datos, normalización y creación de modelo estrella.
Power BI: Modelado dimensional, DAX avanzado y visualización interactiva.

## Desafíos Técnicos Resueltos
Modelado Dimensional: Migración de una tabla plana a un Star Schema para optimizar el rendimiento.
Calidad de Datos: Resolución de inconsistencias en códigos postales y nombres de productos mediante Surrogate Keys.
Análisis YoY (Year-over-Year): Implementación de funciones de inteligencia de tiempo en DAX para comparar el rendimiento anual.

## Hallazgos Clave (Insights)
Rentabilidad Crítica: Se identificó que la categoría de Mesas (Tables) genera pérdidas constantes a pesar de su alto volumen de ventas.
Geografía: Los estados de Texas y Pennsylvania presentan los márgenes más bajos debido a una estrategia agresiva de descuentos.
Crecimiento: El negocio mostró un crecimiento de ventas del $29.5\%$ en el último año analizado.
